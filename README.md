# Salesforce in Power BI

## Description
This project integrates Salesforce data with Microsoft Power BI for analytics and reporting.

## Prerequisites
- Power BI Desktop
  
## PBIT Template Demo
Power BI templates allow for the overall report structure to be saved without attaching any data.  
This demo uses .csv files in this repository as a web data source.

### Instructions for demo
1. Download the pbit file
2. Select "Anonymous"
3. Explore and transform the data in Power Query and within Power BI Desktop
