# Salesforce in Power BI

## Description
This project integrates Salesforce data with Microsoft Power BI for analytics and reporting.

## Features
- Connect Salesforce data sources to Power BI
- Create interactive dashboards and reports
- Automate data refresh schedules
- Visualize key metrics and KPIs

## Prerequisites
- Power BI Desktop or Power BI Service subscription
- Salesforce account with API access
- Required connectors and authentication credentials

## Installation
1. Clone this repository
2. Open Power BI and connect to your Salesforce instance
3. Import the provided templates and dashboards
4. Configure data refresh settings

## Usage
1. Launch Power BI
2. Load the dashboards from this project
3. Customize visualizations as needed
4. Schedule automatic data refreshes
